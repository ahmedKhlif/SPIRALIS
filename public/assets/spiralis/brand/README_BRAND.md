# SPIRALIS Brand Assets v2

These assets were corrected to match the product prototype more closely.

## Use
- `logo-dark.svg`: header/logo on cream or light backgrounds.
- `logo-light.svg`: logo on dark green backgrounds.
- `logo-stacked-dark.svg`: hero/footer variant on light backgrounds.
- `logo-stacked-light.svg`: hero/footer variant on dark backgrounds.
- `logo-mark-dark.svg`: small brand mark, loader, icon.
- `logo-mark-light.svg`: small brand mark on dark backgrounds.
- `favicon.svg`: browser favicon.

## Notes
- Main ink color: `#24513E`
- Dark background: `#173A2A`
- Cream/light color: `#FFF8EC`
- The wordmark is built with editable SVG text using `EB Garamond SC` fallback.
- For exact production branding, load EB Garamond in the website or convert the text to outlines in Illustrator/Figma.
